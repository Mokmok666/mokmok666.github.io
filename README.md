# Wenliang Mo Resume Website

This is a GitHub Pages deployment package for Wenliang Mo's personal resume website.

## Files

- `index.html`: website homepage
- `thesis.pdf`: master's thesis PDF, linked from the education section
- `.nojekyll`: keeps GitHub Pages from applying Jekyll processing
- `DEPLOYMENT_CHECKLIST.md`: upload and verification checklist

## Deployment

Upload all files in this folder to the root directory of the GitHub repository `mokmok666.github.io`.

Do not upload this folder itself or the ZIP file. Upload the extracted files directly.
